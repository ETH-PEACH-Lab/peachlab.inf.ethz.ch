# ETH PEACH Lab Website (Trial Deployment)

This is a **trial deployment** of the ETH PEACH Lab website using Next.js 14 on **GitHub Pages**.

## 🌍 Live Website
The test deployment is available at:  
🔗 [https://eth-peach-lab.github.io/peachlab.inf.ethz.ch/](https://eth-peach-lab.github.io/peachlab.inf.ethz.ch/)

## 🛠️ Setup & Deployment
To run locally:
```sh
git clone https://github.com/ETH-PEACH-Lab/peachlab.inf.ethz.ch.git
cd peachlab.inf.ethz.ch
npm install
npm run dev
```

## Deployment
Once changes are made, view the GitHub page for testing. If looks good, please ping April to make the deployment on [peachlab.inf.ethz.ch](https://peachlab.inf.ethz.ch).
